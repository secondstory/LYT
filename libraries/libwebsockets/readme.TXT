How to setup this library

Plese copy the libs in : arduino-1.5.3\hardware\tools\x86\i586-poky-linux-uclibc\lib
Please replace the platform.win.txt in hardware\arduino\x86 by the one in this folder (revert to the normal one if you don't use the lib)

Please copy the libwesockets.so file on the board, in /usr/lib/
